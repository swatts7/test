
# About
Boilerplate for Material-UI with TS + Storybook + NextJS


## Project Setup

- Nextjs
- TypeScript
- MUI V5
- Storybook
- emotion
- Apollo GraphQL


## Build and Start

```bash
yarn
yarn dev
```

## Storybook

```bash
yarn storybook
```

## Typescript Coverage Report

```bash
yarn ts-coverage
```
